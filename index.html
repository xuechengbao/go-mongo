Go-Mongo

Go-Mongo is a MongoDB driver for the Go programming language.

Installation

Go-Mongo requires a working Go development environment. The Getting Started document describes how to install the development environment. Once you have Go up and running, you can install Go-Mongo with a single command: 

goinstall github.com/garyburd/go-mongo

The Go distribution is Go-Mongo's only dependency.
